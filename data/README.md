# Data Folder

This folder stores runtime data for the bot.

## Files

### whitelist.json
Auto-generated file that stores whitelisted group IDs.
Created automatically when you run `/startmon` command in a group.

**Format:**
```json
{
  "group_ids": [
    -1001234567890,
    -1009876543210
  ]
}
```

**Note:** This file is gitignored to keep your group IDs private.
